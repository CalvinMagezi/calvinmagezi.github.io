Frequently Asked Questions

  • What is a Static Website?

    A static website is one in which there is little to no user interaction and more information. For example an information based website that just allows visitors to view information but not have to login, signup up or comment on any of the information.

  • What is a web application?

    A web application is a website that consists of login, signup, commenting, liking and features of that sort. Web applications take more time and effort to create as they incorporate a lot more Javascript functions than a static website.

  • How long until I see a demo of my site?

    Using a tool called Figma, I am able to draw up a preview of your website in a fairly short amount of time. Combining this with github pages, you will be able to receive a demo link to your static website within 2 days max and watch me as I continue to work on it. Web applications may take more time though.

  • Is Web Design Easy?

    As easy as I may make designing a website easy, it actually is fairly difficult. Luckily I have been given the right tools and training to work efficiently and produce quality products, but it still isn't relatively easy.

  • How do you price a website?

    Website development is priced according to the time and effort put into each product. I do have a price cap on website and web app development so that even though you have a challenging website I won't over price you for the time it takes to develop it.
